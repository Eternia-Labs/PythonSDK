# PythonSDK

## Setup steps

#### 1. Clone repo
#### 2. activate virtualenv and cd to folder PythonSDK
#### 3. Build a wheel - python3 setup.py sdist bdist_wheel
#### 4. pip install -e /path/to/the/root_folder/containing/setup.py 
#### 5. Test in venv
